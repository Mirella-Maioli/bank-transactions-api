# Bank Transactions API

REST API for managing bank accounts and financial transactions built with Python and FastAPI.

This project simulates a simple banking backend where users can create accounts, check balances and perform financial transactions.

## Features

- Create bank accounts
- Check account balance
- Register deposits and withdrawals
- List transaction history

## Technologies

- Python
- FastAPI
- REST API
- Backend development

## Endpoints

GET /accounts  
POST /accounts  

GET /balance/{account_id}  

POST /transactions  
GET /transactions
